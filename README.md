# Kevin's Place

Personal website and blog by Kevin (Wenjie) Guo.

**Live site:** [https://kevin-wenjie-guo.github.io/page/](https://kevin-wenjie-guo.github.io/page)

## Sections

| Section | Description |
|---------|-------------|
| Daily Writing | Posts, reflections, and things I'm learning |
| Investing | Equity research, investment theses, market thinking |
| App Dev | Applications I'm building and software learnings |
| Strategy | Business strategy and management thinking |
| Meditation | Mindfulness, well-being, and living better |

## Built With

- [Quarto](https://quarto.org) — open-source publishing system
- Deployed via GitHub Pages with GitHub Actions

## Local Development

```bash
# Install Quarto: https://quarto.org/docs/get-started/
quarto preview
```

## Adding a New Post

Create a folder inside the relevant section directory with an `index.qmd`:

```
investing/my-new-post/index.qmd
```

Push to `master` and the site will auto-deploy.
